# Julie Ray Real Estate
Modern, responsive website design for Julie Ray built with React + Typescript. The site features dynamic background images, a location search form, and a clean, user-friendly interface for browsing and discovering real estate properties.

### Features

### Tech Stack
* React – Frontend UI library
* TypeScript – Type safety and better developer experience
* Vite – Fast build tool and dev server
* CSS / Flexbox – Layout and animations
* OpenStreetMap Nominatim API for location search

### Getting Started
#### Requirements
* Node.js >= 18
#### Install dependencies
```bash
npm install
```
#### Run development server
```bash
npx vite
```
#### Build and run for production 
```bash
npm run build
npm run preview
```
